# Le plus petit agent IA possible — pi Agent (l'agent derrière OpenClaw)

Dans cette vidéo on réagit à la vidéo "Building PI in a Word of Slop" : https://www.youtube.com/watch?v=RjfbvDXpFls

pi.dev (https://pi.dev/) est un agent minimal qui contient 4 tools et super extensible. La vidéo explique comment pi a été créé et la philosophie derrière qui nous propose de ralentir dans notre approche à l'AI mais aussi de comprendre un peu plus ce qu'il se passe.
